# 🗂 タブ切替（コレクション切り替えUI）

複数のコレクションや商品グループを「タブ」で切り替え表示するUI。
ユーザーの回遊性を高めつつ、1画面内に情報を整理して表示できます。



https://github.com/user-attachments/assets/862fd7e4-50f5-4e26-ae4a-35fb02ebe383



📌 ポイント

✅ 同一画面で商品カテゴリを切り替え可能

✅ ページ遷移が不要のため離脱を防止

✅ ギフトECやカテゴリ数の多い商材と相性が良い

✅ PC / SP 両方で使いやすいUIにできる

✅ 商品一覧・ランキング・特集など応用範囲が広い

🎯 ユースケース例

✅ カテゴリ切替（例：バレンタイン / ホワイトデー / ギフト）

✅ 属性切替（例：セット / 単品 / 人気順）

✅ 季節訴求（例：春ギフト / 夏ギフト / お歳暮）

✅ ランキング軸切替（例：総合 / 男性向け / 女性向け）

✅ 実際にギフトECでは「悩んでいる→比較→購入」導線が自然になるためCVR改善に繋がります。
